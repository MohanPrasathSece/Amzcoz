# AMZCOZ Portfolio Website

Modern, minimal, and creative portfolio website for AMZCOZ - Amazon Growth Experts.

## Features

- Modern React application with Vite
- Smooth animations with Framer Motion
- Responsive design with light blue and white theme
- Interactive components and hover effects
- Lead generation forms and modals
- Multi-page routing with React Router

## Tech Stack

- React 18
- Vite
- React Router DOM
- Framer Motion (animations)
- React Icons
- React Intersection Observer

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
